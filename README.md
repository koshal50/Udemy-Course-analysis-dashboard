# Udemy Course Analysis Dashboard (Power BI)

A Power BI dashboard for analyzing Udemy course performance and sales trends. It provides a high-level summary of key metrics and interactive visuals to explore revenue patterns, learner demand, and course characteristics.

## What’s Inside

### Key KPIs (Top Cards)
- **Total Revenue**
- **Total Students**
- **Total Courses**
- **Average Price**

### Interactive Analysis
- **Quarter slicer** to filter performance by time period  
- **Payment Type slicer** to compare transaction preferences  
- **Subject-wise revenue distribution** (bar chart) to compare categories  
- **Course level insights** (Beginner / Intermediate / All Levels) to understand learner demand  
- **Lectures vs Duration** (scatter plot) to explore relationships between course structure and time

## User Manual (How to Use the Dashboard)

1. Open the dashboard in **Power BI Desktop** or **Power BI Service** to view the main summary page.
2. Review the **KPI cards at the top** to understand total revenue, students, courses, and average price.
3. Use the **Quarter slicer** to filter data based on specific time periods.
4. Select different **Payment Types** from the slicer to analyze transaction preferences.
5. **Hover** over charts and graphs to view detailed tooltips and exact values.
6. **Click** on any bar, pie slice, or data point to apply **cross-filtering** across the dashboard.
7. Analyze **subject-wise revenue distribution** using the bar chart for category comparison.
8. Explore **course levels** (Beginner, Intermediate, All Levels) to understand learner demand.
9. Use the **scatter plot** to examine the relationship between **lectures** and **course duration**.
10. **Reset filters / clear selections** to return to the default dashboard view for a fresh analysis.
